Syllabus Overview

Day 1: Foundations & Frontend Magic

  • Product Management Essentials: From idea to PRD (Product Requirements Document).
  • UI/UX Best Practices: Designing user-centric interfaces with Figma.
  • Version Control with Git: Mastering branching, commits, and pushing to GitHub.
  • Vibe Coding Example #1: Build a landing page with checkout & an integrated AI chatbot.
  • Vibe Coding Example #2: Develop a functional Google Chrome plugin from scratch.

Day 2: Backend, Deployment & Going Live

  • Backend & Database Integration: Connecting your app to serverless functions and databases.
  • Live Deployment: Pushing your projects live with proper hosting on Vercel/Netlify.
  • Vibe Coding Example #3: Create a cross-platform mobile app for iOS and Android.
  • Vibe Coding Example #4: Build a thematic project with real-world application (e.g., e-commerce, social media clone).
